# 16th Hihat

## 是什么

`16th hihat` 指的是 hi-hat 以十六分音符持续铺满一整小节。

## 听感

它会明显提升速度感、密度感和前进感，是很多 house、techno、disco 和 club groove 的基础表层。

## Strudel 示例

```js
$: s("hh*16").bank("RolandTR909")
```

这里的 `*16` 表示每个 cycle 播放 16 次，所以会形成连续的十六分 hat。

## 使用提醒

- 很适合叠在稳定的 `4-4 kick` 上。
- 如果已经有很多细碎 percussion，就要小心不要太满。
