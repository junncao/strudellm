# Triplet Pulse

## 是什么

`Triplet pulse` 指的是三连感驱动的基础脉冲，而不是标准的 straight grid。

## 听感

它会让节奏更滚动、更摆、更像在“往前卷”，常见于 shuffle、swing、更黑一点的 groove 里。

## Strudel 示例

```js
$: s("bd*3").slow(2).bank("RolandTR909")
```

这个例子在更长的时间里放 3 个均匀脉冲，会让听感偏向 triplet / 12 feel。

## 使用提醒

- 很适合和 `shuffle`、`laid-back` 一起使用。
- 如果 bass 也走三连，会更明显地建立这种语法。
